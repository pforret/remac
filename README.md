# remac

change the MAC address of your computer to avoid trackability

## Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/remac

or with `git`

	$ git clone https://github.com/pforret/remac.git
	$ cd remac

## Usage

	> remac.sh [action]
